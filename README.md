
1. Install Xcode
AppStoreから手動でインストールする。

2. dotfileリポジトリをgit cloneしてくる。
・git clone 実行するといくつかヒアリングがあったはず。

3. setup_dotfile.shを実行する
・bashrcとかdotfileがセットされる
4. install.shを実行する
・ HomeBrewがインストールされる

5. brew bundle を実行する
・Brewfileの中身がInstallされる

6. Macの初期設定
・Mac>システム環境設定>キーボード
　　キーボード>　F1,F2などのすべてのキーを標準のファンクションキーとして使用を有効にする

7. MacApplication
